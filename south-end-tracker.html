📱 iPhone / iPad Safari — 3 layers must all be on: 1. Settings → Privacy & Security → Location Services → ON

🤖 Android Chrome Tap the 🔒 in the address bar → Permissions → Location → Allow.
💻 Desktop Chrome / Edge Tap 🔒 in the address bar → Site settings → Location → Allow.
🦊 Firefox Tap 🔒 → Connection secure → More info → Permissions → Location → Allow.
